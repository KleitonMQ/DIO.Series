Projeto proposto pelo bootcamp GFT Start #5

Descrição:

Aplicação feita para console em C#. Apresenta um menu com opções de controle de cadastro, visualização geral e específica de séries.
Objetivo principal do projeto foi, praticar os conceitos de Programação Orientada a Objetos (utilizando Classes, Interface, Enum, Herança) e Tratamento de exceções.
